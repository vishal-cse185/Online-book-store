# Online Bookstore Management System

A Java Spring Boot web application developed for managing online bookstore operations efficiently. This project demonstrates CRUD operations, MVC architecture, and database integration using Spring Boot, Thymeleaf, Hibernate JPA, and MySQL.

## Features
- Add Books
- View Books
- Book Data Management
- MVC Architecture
- Database Integration

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- Hibernate JPA
- MySQL
- Maven

## Project Structure

src/main/java
- controller
- model
- repository

src/main/resources
- templates
- application.properties
